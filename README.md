# hookify

Легкий реєстр плагінів з хуками `before_request` / `after_response` для обгортання FastAPI та інших сервісів. Використовується в [FastLM-API](https://github.com/kiurakku/FastLM-API).

## Встановлення

```bash
git clone https://github.com/kiurakku/Hookify.git
cd Hookify
pip install -e .
```

## Git

Публікуйте **першим** серед трьох pet-проєктів, щоб збірка FastLM могла встановити пакет з GitHub:

```bash
cd /шлях/до/Hookify
git add -A
git commit -m "Опис змін"
git remote -v
git push -u origin main
```
